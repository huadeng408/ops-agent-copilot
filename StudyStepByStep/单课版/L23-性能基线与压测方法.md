# L23 性能基线与压测方法

## 本课定位
建立“可重复、可比较、可解释”的性能基线。

## 图解页
```mermaid
flowchart LR
Case["固定请求集"] --> Load["压测执行"]
Load --> Metric["吞吐/延迟/错误率"]
Metric --> Baseline["形成基线"]
```

## 术语表
- Baseline：性能基线
- Tail Latency：尾延迟
- Throughput：吞吐

## 面试问题与标准答案
1. 为何看P95/P99？  
答案：尾延迟更接近真实用户体验。
2. 客户端可能成为瓶颈吗？  
答案：会，需同时监控压测端资源和服务端指标。
3. 何为有效优化？  
答案：核心指标改善且无明显副作用。
