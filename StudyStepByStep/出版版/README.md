# 出版版总览

本目录是 `StudyStepByStep` 的正式阅读入口，定位不是“零散讲义汇总”，而是可以持续迭代、适合打印和面试训练的出版增强版本。

## 你会在这里看到什么

- `L01-L30`：30 节逐课出版稿，每课都带元信息、图解页、源码锚点、误区提醒、口播模板和参考文献。
- [课程目录](./课程目录.md)：按课次索引整套内容，适合顺序学习和跳转复习。
- [排版规范](./排版规范.md)：统一标题层级、图表、页脚和引用风格。
- [参考文献总表](./参考文献总表.md)：集中查看整套资料涉及的参考来源。
- [出版导出说明](./出版导出说明.md)：说明如何导出 PDF、迁移到 Notion、保留 Mermaid 图和目录结构。
- [版本记录](./版本记录.md)：记录本套资料的结构升级与修订轨迹。

## 推荐使用方式

### 路线一：系统精学

1. 先读 [课程目录](./课程目录.md)，明确 30 节的前后依赖。
2. 按 `L01 -> L30` 顺序学习，每节课读完后立刻完成“课后任务与参考答案”。
3. 学完对应课次后，回到 `../讲师版脚本/` 做 3 到 5 分钟口述复盘。

### 路线二：面试冲刺

1. 先看每课的“60 秒面试口播模板”，建立统一输出框架。
2. 再看“关键源码锚点”和“常见误区”，把回答和源码证据绑定起来。
3. 最后配合 [../7天冲刺手册.md](../7天冲刺手册.md) 做计时训练。

### 路线三：授课/分享

1. 用每课开头的 Mermaid 图做开场图示。
2. 用“实战检查清单”作为课堂验收标准。
3. 用 `讲师版脚本` 做口述提纲，用 `出版版` 做讲义正文。

## 本版和其他目录的关系

- `../01-07/`：分册深度版，适合按主题纵向学习。
- `../单课版/`：单课基础版，是出版版的干净内容源。
- `../讲师版脚本/`：口述训练与授课脚本。
- `../assets/`：统一架构图、请求流图、依赖图和数据流图。

## 这一版的质量标准

- 每课必须能回答“为什么这样设计”，而不只是“它是怎么工作的”。
- 每课必须给出源码锚点，保证面试表达有代码证据。
- 每课必须有误区与检查清单，避免学完只能复述概念。
- 每课必须可独立导出和打印，不依赖上下文才能阅读。

## 下一步怎么用最划算

- 如果你是第一次系统学习，从 [L01-项目定位与边界.md](./L01-项目定位与边界.md) 开始。
- 如果你已经跑通过项目，优先看 `L09-L22`，这是最容易被一面深挖的模块区间。
- 如果你准备把资料沉淀成 PDF 或知识库，先看 [出版导出说明](./出版导出说明.md)。
